# Historia 11 - Ver mis reservas (panel de usuario)

## Descripcion

Como usuario,
quiero consultar mis reservas,
para revisar estado, horario y gestionar acciones.

## Objetivo

Ofrecer una vista clara del historial y reservas activas del usuario.

## Criterios de aceptacion

- Se listan reservas del usuario autenticado.
- Cada reserva muestra maquina, fecha, hora y estado.
- Se permite cancelar cuando aplique.
- Se contemplan estados vacios y errores de carga.

## Tareas de implementacion

- Implementar pagina de Mis reservas.
- Consumir endpoint de reservas del usuario.
- Normalizar datos para visualizacion.
- Integrar acciones de gestion sobre cada reserva.

## Prioridad

Alta.

## Dependencias

Historia 8.

## Estimacion

4 horas.
