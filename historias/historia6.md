# Historia 6 - Visualizacion de maquinas disponibles

## Descripcion

Como usuario,
quiero consultar las maquinas disponibles,
para decidir cual reservar.

## Objetivo

Mostrar un catalogo filtrado por gimnasio y con informacion util para reservar.

## Criterios de aceptacion

- El usuario ve maquinas de su gimnasio.
- El admin puede ver catalogo global.
- Cada maquina muestra nombre, estado, zona e imagen.
- Se puede abrir detalle de maquina.

## Tareas de implementacion

- Construir vista de listado de maquinas.
- Aplicar filtro por gimnasio para usuario.
- Crear tarjeta visual de maquina.
- Enlazar al detalle de cada maquina.

## Prioridad

Alta.

## Dependencias

Historia 5.

## Estimacion

4 horas.
