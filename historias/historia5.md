# Historia 5 - Gestion de maquinas (CRUD completo)

## Descripcion

Como administrador,
quiero crear, editar y eliminar maquinas,
para mantener un catalogo correcto y utilizable.

## Objetivo

Administrar maquinas por gimnasio y su estado operativo.

## Criterios de aceptacion

- Se pueden crear maquinas asociadas a un gimnasio.
- Se pueden editar estado, descripcion e imagen.
- Se pueden eliminar maquinas.
- El listado de maquinas se actualiza tras cada cambio.

## Tareas de implementacion

- Implementar modulo admin de maquinas.
- Validar campos obligatorios y estado permitido.
- Integrar selector de gimnasio.
- Ajustar listado con metadatos de maquina.

## Prioridad

Alta.

## Dependencias

Historia 4.

## Estimacion

5 horas.
