# Historia 4 - Gestion del gimnasio (administrador)

## Descripcion

Como administrador,
quiero gestionar los datos del gimnasio,
para mantener la informacion operativa actualizada.

## Objetivo

Permitir alta, edicion y baja de gimnasios desde el panel de administracion.

## Criterios de aceptacion

- El admin puede crear gimnasios.
- El admin puede editar nombre, direccion y telefono.
- El admin puede eliminar gimnasios.
- El listado refleja cambios en tiempo real de interfaz.

## Tareas de implementacion

- Implementar CRUD de gimnasios en admin.
- Crear formulario con validaciones basicas.
- Conectar operaciones con API.
- Refrescar datos dependientes tras cambios.

## Prioridad

Alta.

## Dependencias

Historia 1 y 3.

## Estimacion

4 horas.
